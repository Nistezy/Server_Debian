Servidor DHCP (IPv4 e IPv6) – CypherBaron Lab
Este repositório documenta a configuração do servidor DHCPv4 e DHCPv6 utilizado no ambiente de estudos CypherBaron Lab, baseado em Debian. O objetivo é fornecer um ambiente realista para aprendizado de redes, segurança, infraestrutura e testes de ataques/defesas.

Este laboratório utiliza um servidor Debian configurado para fornecer: DHCPv4 – distribuição automática de endereços IPv4 DHCPv6 – distribuição automática de endereços IPv6 Integração com DNS interno
