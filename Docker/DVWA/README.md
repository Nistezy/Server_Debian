DVWA – Damn Vulnerable Web Application
Este repositório contém a configuração necessária para subir o ambiente DVWA utilizando Docker + Docker Compose.
O DVWA é uma aplicação web intencionalmente vulnerável, usada para estudos e testes de segurança ofensiva.

Objetivo do Projeto
Criar um ambiente seguro e isolado para estudar:
OWASP Top 10
SQL Injection
XSS
Command Injection
CSRF
File Inclusion
Autenticação insegura
E diversas outras vulnerabilidades reais
Tudo isso sem colocar em risco o restante do servidor ou da rede local.
