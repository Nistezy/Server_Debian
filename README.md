# Server_Debian
Este repositório documenta a configuração, manutenção e evolução de um servidor baseado em Debian, criado com foco em estudos de cibersegurança, administração de sistemas e simulações de ambientes reais.  O servidor foi projetado para oferecer um ambiente completo para aprendizado, contando com serviços essenciais para análise, ataque e defesa.
