Servidor DNS (BIND9) – CypherBaron Lab
Este documento orienta a configuração, validação e testes de um servidor DNS autoritativo + recursivo utilizando o BIND9 em sistemas baseados em Linux (Debian). Inclui suporte a: 
Zona Forward (FQDN → IP) 
Zona Reverse (IP → FQDN) 
Encaminhamento de consultas externas (Forwarders) 
DNSSEC Validation Verificação de configuração 
Testes com dig e nslookup
